# LabsWork - Landing Page 🔬

Una landing page moderna, responsiva y orientada a la conversión para un Laboratorio de Reparación de Precisión y Equipos de Alta Tecnología. 

El sitio está diseñado con un enfoque técnico y minimalista (Dark Theme), destacando los servicios especializados, la metodología de trabajo y facilitando el contacto directo con los clientes a través de un formulario funcional y un enlace a WhatsApp.

## 🚀 Características Principales

* **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y pantallas de escritorio.
* **Formulario de Contacto Funcional:** Integrado con Web3Forms para enviar consultas directamente al correo sin necesidad de un backend propio.
* **Validación de Datos:** Campos obligatorios con mensajes de error personalizados en español.
* **Llamados a la Acción (CTA):** Enlace directo a WhatsApp para asistencia instantánea.
* **Interfaz de Usuario (UI):** Estética "tech" utilizando Tailwind CSS, tipografías monoespaciadas y variables CSS personalizadas.

## 🛠️ Tecnologías Utilizadas

* **HTML5** semántico.
* **Tailwind CSS** (vía CDN para un prototipado rápido).
* **CSS3** (Variables globales para el esquema de colores y scrollbar personalizado).
* **Web3Forms** (API para el procesamiento del formulario de contacto).
* **Google Fonts:** `Inter` (texto general) y `JetBrains Mono` (acentos técnicos).
* **Material Symbols Outlined** (Iconografía).
